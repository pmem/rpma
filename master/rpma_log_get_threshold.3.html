NAME

rpma_log_get_threshold - get the logging threshold level

SYNOPSIS

#include <librpma_log.h>

int rpma_log_get_threshold(rpma_threshold threshold, rpma_log_level *level);

DESCRIPTION

rpma_log_get_threshold() gets the current level of the threshold. See rpma_log_set_threshold(3) for available thresholds and levels.

RETURN VALUE

rpma_log_get_threshold() function returns 0 on success or error code on failure.

ERRORS

rpma_log_get_threshold() can fail with the following errors:

•

RPMA_E_INVAL - threshold is not RPMA_LOG_THRESHOLD_PRIMARY nor RPMA_LOG_THRESHOLD_SECONDARY

•

RPMA_E_INVAL - level is NULL
