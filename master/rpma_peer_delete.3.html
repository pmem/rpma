NAME

rpma_peer_delete - delete a peer object

SYNOPSIS

#include <librpma.h>

int rpma_peer_delete(struct rpma_peer **peer);

DESCRIPTION

RETURN VALUE

The rpma_peer_delete() function returns 0 on success or a negative error code on failure. rpma_peer_delete() does not set *peer to NULL on failure.

ERRORS

rpma_peer_delete() can fail with the following error:

•

RPMA_E_PROVIDER - deleting the verbs protection domain failed.
