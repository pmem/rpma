NAME

rpma_mr_get_descriptor - get a descriptor of a memory region

SYNOPSIS

#include <librpma.h>

int rpma_mr_get_descriptor(struct rpma_mr_local *mr,
struct rpma_mr_descriptor *desc);

DESCRIPTION

rpma_mr_get_descriptor() writes a network-transferable description of the provided local memory region. Once the descriptor is transferred to the other side it can be consumed by rpma_mr_remote_from_descriptor() to create a remote memory region’s structure which allows transferring data between the peers.

ERRORS

rpma_mr_get_descriptor() can fail with the following error:

•

RPMA_E_INVAL - mr or desc is NULL
