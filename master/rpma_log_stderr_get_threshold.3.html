NAME

rpma_log_stderr_get_threshold - get the current log level to stderr threshold

SYNOPSIS

#include <librpma_log.h>

rpma_log_level rpma_log_stderr_get_threshold(void);

DESCRIPTION

rpma_log_stderr_get_threshold(3) gets the current log level to stderr(3) threshold.

RPMA_LOG_DISABLED indicates that writing to stderr(3) is disabled.

See rpma_log_syslog_set_threshold(3) for available thresholds.

RETURN VALUE

rpma_log_stderr_get_threshold() returns the actual threshold for logging to stderr(3) or RPMA_LOG_DISABLED if logging to stderr(3) is disabled.
